Reducing AI Hallucinations

Reduce AI-generated errors and fabricated data in your work

The generic prompt
"Summarise the key findings from this report."
The structured prompt View full prompt ▼
"Summarise the key findings from the attached report. Only use information explicitly stated in the document..."
Summarise the key findings from the attached report. Only use information explicitly stated in the document. For each finding, cite the page or section number. If a data point is referenced but the source value is not in the document, state "not available in source." Do not infer, estimate, or generate any statistics not present in the original.
Generic output
  • Confident claims with no source references
  • Statistics that don't appear in the original
  • No way to verify what's real and what's fabricated
Structured output
  • Every claim cites a page or section number
  • Missing data clearly flagged as "not available"
  • Verifiable output you can trust and share
Unverifiable → Trustworthy

Budget Variance Analysis

The generic prompt
"Analyse the budget variances."
The structured prompt View full prompt ▼
"Analyse Q4 budget variances. For any line item >5% over or under budget..."
Analyse Q4 budget variances. For any line item >5% over or under budget: 1) Calculate variance amount and %, 2) Identify trend (one-time vs ongoing), 3) Flag items requiring immediate action. Format as table with: Department, Line Item, Variance $, Variance %, Trend, Action Required.
Generic output
  • Generic summary of numbers
  • No context on why variances occurred
  • Still requires manual investigation
Structured output
  • Structured table, ready for leadership
  • Variances prioritised by impact
  • Clear action flags for follow-up

Performance Review Summary

The generic prompt
"Summarise this employee's performance."
The structured prompt View full prompt ▼
"Review this employee's performance notes from the past year. Create a summary structured as..."
Review this employee's performance notes from the past year. Create a summary structured as: 1) Key Accomplishments (3-5 specific examples with metrics), 2) Areas for Development (2-3 items with context), 3) Goals Progress (% completion on each stated goal), 4) Recommended Rating (Exceeds/Meets/Below with brief justification). Keep tone professional and constructive.
Generic output
  • Vague, generic summary
  • No specific accomplishments or metrics
  • Requires complete rewrite
Structured output
  • Structured with specific examples and metrics
  • Clear rating with justification
  • Manager reviews and personalises in minutes
